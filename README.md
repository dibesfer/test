## Workflow law

GitHub repo > committed files > source of truth.

Local files are invisible until pushed.
GitHub Pages is preview, not truth.
Small commits beat giant mystery patches.

## Fer + Alexos protocol

1. Fer gives the goal.
2. Alexos reads the repo.
3. Alexos changes the smallest useful thing.
4. Alexos commits.
5. Fer reviews, tests, and steers.
